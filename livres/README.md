# Livres Tech — Recommandations (mini-site)

Simple page statique listant des livres tech recommandés.

Fichiers créés:

- index.html
- styles.css
- script.js

Tester localement:

Ouvrez `index.html` dans un navigateur, ou servez le dossier avec un serveur local, par exemple:

```bash
cd ~/book-recommendations
python3 -m http.server 8000
# puis ouvrez http://localhost:8000
```

Voulez-vous que je :

- initialise un dépôt Git pour ce dossier ?
- pousse le site sur GitHub Pages ?
